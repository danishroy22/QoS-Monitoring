# Development Roadmap

## Phase 1: Architecture and Folder Structure

Duration: Week 1-2

Deliverables:

- System architecture diagram
- Folder structure
- Database schema
- API contract
- Simulation scenario definitions
- Testing methodology

## Phase 2: Network QoS Data Simulator

Duration: Week 3-5

Deliverables:

- Python QoS measurement generator
- Normal and abnormal traffic scenarios
- Configurable virtual broadband nodes
- CSV export and API publishing mode
- Ground-truth event labels for ML evaluation

## Phase 3: Backend API and Database

Duration: Week 6-8

Deliverables:

- FastAPI application
- PostgreSQL connection layer
- SQLAlchemy models
- Measurement ingestion endpoint
- Historical metrics endpoint
- Latest network health endpoint

## Phase 4: Machine Learning Anomaly Detection

Duration: Week 9-11

Deliverables:

- Feature engineering pipeline
- Isolation Forest baseline model
- Anomaly scoring endpoint
- Severity classification
- Evaluation using simulator ground truth

## Phase 5: Dashboard

Duration: Week 12-14

Deliverables:

- React dashboard
- Live QoS metric cards
- Historical Chart.js graphs
- Network health status
- Detected issues panel
- Responsive layout

## Phase 6: Generative AI Analysis

Duration: Week 15-17

Deliverables:

- AI prompt templates
- Explanation endpoint
- Recommendation endpoint
- Offline fallback explanation generator
- Dashboard integration

## Phase 7: Testing, Evaluation, and Dissertation

Duration: Week 18-20

Deliverables:

- Unit and integration tests
- ML evaluation results
- Dashboard screenshots
- Architecture diagrams
- Final dissertation chapters
- Demonstration script
